# Uber Analytics Dashboard (Power BI)

## Project Overview

This project is an **Uber Analytics Dashboard** developed using **Microsoft Power BI** to analyze real-world Uber trip data.
The dashboard helps visualize booking performance, revenue trends, vehicle-wise analysis, trip distance, and rider and driver ratings to support business decision-making.

## Business Problems & Requirements

The dashboard is designed to address key Uber business questions, including:

* How many bookings are successfully completed or lost?
* How revenue and bookings change over time (monthly and quarterly)?
* Which vehicle types generate the highest revenue?
* Which pickup and drop locations have the highest demand?
* How riders and drivers rate the service experience?

## Key Performance Indicators (KPIs)

* Completed Bookings
* Lost Bookings
* Total Revenue
* Total Distance
* Average Distance per Trip

## Dashboard Features

* **Overview Page** with key KPIs and insights
* **Vehicle Type Filter** for detailed analysis
* **Monthly Analysis**

  * Completed Bookings
  * Revenue
* **Quarterly Analysis**

  * Completed Bookings
  * Revenue
* **Revenue by Vehicle Type**
* **Top Pickup Locations** based on booking count
* **Top Drop Locations** based on booking count
* **Ratings Analysis**

  * Average Rider Ratings
  * Average Driver Ratings

1. Open the `Uber_Analytics_Dashboard.pbix` file in Power BI Desktop.
2. Navigate through the Home and Overview pages.
3. Use vehicle filters and time-based analysis to explore booking and revenue trends.
4. Interact with visuals to gain deeper operational insights.

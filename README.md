# Uber Analytics Dashboard (Power BI)

## Project Overview
This project is an **Uber Analytics Dashboard** built using **Power BI** to analyze real-world Uber trip data.  
It helps visualize key metrics like ride trends, cancellations, bookings, and driver performance.  

## Files Included
- `Uber_Analytics_Dashboard.pbix` : Power BI project file  
- `Home_Page_Uber_Dashboard.png` : Screenshot of the home page  
- `Overview_Page_Uber_Dashboard.png` : Screenshot of the overview page  
- `Uber_Analytics_Dashboard.pdf` : Detailed project report  

## Key Features
- Interactive dashboards for better decision-making  
- Visualizations for ride statistics, rider & driver cancellations  
- Booking trend analysis over time  
- Key Performance Indicators (KPIs) displayed for quick insights  

## Technologies Used
- **Power BI** for dashboard and data visualization  
- **DAX** for data analysis expressions (if applicable)  
- **Real-world Uber dataset**  

## How to Use
1. Open the `Uber_Analytics_Dashboard.pbix` file in Power BI Desktop.  
2. Navigate through Home and Overview pages to explore visualizations.  
3. Click on charts to filter and drill down insights.
